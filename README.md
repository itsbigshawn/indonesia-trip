# You, me + Indonesia? 🌴

A simple one-page interactive map of our Indonesia girls' trip, built with
[Leaflet](https://leafletjs.com/) + OpenStreetMap. No build step, no API keys —
just open `index.html`.

## The route

**Bali → Flores → Jogjakarta → Jakarta** · 14 nights of pure adventure ✨

| # | Stop | Nights | Highlights |
|---|------|--------|------------|
| 1 | 🌺 Bali | 4 | Ubud temples & rice terraces, Mount Batur, Sanur beaches |
| 2 | 🐉 Flores (Labuan Bajo) | 4 | Liveaboard, Komodo, Pink Beach, Padar sunrise, manta rays |
| 3 | 🛕 Jogjakarta | 3 | Borobudur & Prambanan, museums, batik, Malioboro |
| 4 | 🏙️ Jakarta | 3 | Old Town, Glodok Chinatown, food & café hopping |

## Run locally

Just open `index.html` in a browser, or:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Published with GitHub Pages

This site is served straight from the repository root via GitHub Pages.
